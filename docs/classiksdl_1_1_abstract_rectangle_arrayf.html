Detailed Description

Gathers the common code related to the drawing of rectangle arrays using float coordinates.

This is the recommended approach when drawing several rectangles. It is faster than drawing all the rectangles individually, since only one single call is made to draw all the rectangles.

Warning
All the rectangles in the array are drawn with the same color
