Detailed Description

template<DrawablePosition T>
class iksdl::BaseLineArray< T >

An array of connected lines that can be drawn.

This is the recommended approach when drawing several connected lines. It is faster than drawing all the lines individually, since only one single call is made to draw all the lines.

Warning
All the lines in the array are drawn with the same color
