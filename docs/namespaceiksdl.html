template<typename K , typename V >
bool iksdl::comparePairKeys ( const std::pair< K, V > &  p1,
const std::pair< K, V > &  p2 
)
inline

Compares two instances of std::pair where the first component behaves like a primary key.

The default implementation for the comparison of two instances a and b of std::pair is something like:

return a.first < b.first || (a.first == b.first && a.second < b.second);

However, if we are sure that first element is a primary key, then a.first and b.first cannot be equal. Therefore, in this case the default implementation may lead to useless comparisons.

This function can be used to avoid useless computations in the case described above.

Parameters
p1First pair to compare
p2Second pair to compare
Returns
True if the key of p1 is lower than the key of p2
