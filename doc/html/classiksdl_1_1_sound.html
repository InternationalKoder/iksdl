IKSDL_EXPORT void iksdl::Sound::play ( std::chrono::milliseconds  time,
int  loops = 0 
)

Play the sound for a limited amount of time.

By default the sound is played once (fully or partially). Set the loops parameter to change this behaviour. When loops is set to -1, the sound will be infinitely repeated until the time limit is reached.

Parameters
timeMaximum time to play the sound
loopsMaximum number of times to play the sound
